Regra de Commits para organização

Documentação:
git commit -m "doc: fiz X no doc X"

Bug em código:
git commit -m "bug: arrumei bug X em X"
git commit -m "fix: arrumei X em X"

Nova funcionalidade:
git commit -m "feature: adicionei X em X"

Estilização:
git commit -m "style: arrumei X em X"

Atualização:
git commit -m "update: atualizei X em X"
